# Catalog Snapshot

## Product Categories

The Northwind catalog includes the following categories:

- **Beverages:** Soft drinks, coffee, tea, juices
- **Condiments:** Spices, sauces, seasonings
- **Confections:** Candies, desserts, sweets
- **Dairy Products:** Milk, cheese, yogurt
- **Grains/Cereals:** Breads, cereals, pasta
- **Meat/Poultry:** Fresh and prepared meats
- **Produce:** Fresh fruits and vegetables
- **Seafood:** Fish, shellfish, and other seafood

All products map to these categories as defined in the Northwind database schema (Products.CategoryID).
